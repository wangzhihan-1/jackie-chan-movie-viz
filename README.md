# 拳脚与镜头：成龙电影数据可视化分析
## 项目介绍
本项目基于D3.js开发成龙六十年电影生涯多维度数据可视化网页，黑金胶片电影复古风格，包含十大动态交互图表，鼠标悬浮查看详细数据，分四大章节切换浏览。
数据维度：电影类型谱系、时序产量票房、主创合作网络、全球市场热力、视听剪辑语言量化。

## 技术栈
- HTML5 + Tailwind CSS 页面布局
- D3.js v7 全套可视化图表（树图、环形图、直方图、力导向图、等高线等）
- GitHub Pages 免费在线部署，公开分享链接

## 功能说明
1. 顶部导航按钮切换四大分析页面，无刷新切换
2. 全部图表支持鼠标悬浮弹窗展示细分数据
3. 整体黑金鎏金电影胶片视觉风格，摄影机、胶片齿孔装饰元素
4. 完整匹配课程作业文档全部分析维度与图表逻辑

## 本地运行
1. 下载项目文件
2. 直接双击index.html打开浏览器即可运行，无需本地服务
